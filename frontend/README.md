# Farmly Frontend

## Description
The frontend user interface for Farmly, built with React and Vite. It provides a responsive experience for farmers to access all platform features.

## Prerequisites
- Node.js (v14+)

## Environment Variables
Create a `.env` file in this directory:
```env
VITE_API_URL=http://localhost:5000/api/v1
```
*Note: If running on mobile, replace localhost with your machine's local IP address.*

## Running
```bash
npm install
npm run dev
```
Access the application at `http://localhost:5173`.
